# Struktur Frontend Database Yayasan

Folder `src` menjadi pusat kode aplikasi agar file konfigurasi root tetap bersih.

- `app`: routing utama Next.js App Router.
- `components`: komponen UI yang bisa dipakai ulang.
- `hooks`: React hooks untuk kebutuhan lintas halaman.
- `lib`: helper, API client, auth helper, formatter, dan constant.
- `types`: tipe data TypeScript yang mengikuti response dari backend.
